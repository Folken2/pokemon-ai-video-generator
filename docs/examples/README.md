# Example Pokemon Documentaries

This directory contains example outputs from the Pokemon Natural Geographic pipeline.

| Pokemon   | Status        | Description                                    |
|-----------|---------------|------------------------------------------------|
| haunter   | Complete      | Full pipeline: research → story → assets → video |
| bulbasaur | In progress   | Bulbasaur vs Spearow survival story            |
| charizard | In progress   | Charizard evolution and dominance              |
| pikachu   | In progress   | "First Spark" - juvenile Pikachu and Fearow    |

Use these as reference when creating your own documentaries. For new projects, create a folder at the project root (e.g., `squirtle/`) and follow the prompts in `prompts/`.
